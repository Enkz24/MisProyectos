Sistema POS para Farmacias y Mayoristas

Sistema de punto de venta adaptado completamente para farmacias y tiendas mayoristas de abarrotes, enfocado en control de productos, fechas de caducidad, compras y gestión de precios.

Descripción del Sistema

Este sistema fue desarrollado tomando como base un POS inicial en PHP, el cual fue reestructurado y ampliado para cubrir necesidades específicas del rubro farmacéutico y de comercios mayoristas.

El enfoque principal estuvo en el control de fechas de vencimiento, alertas preventivas, gestión de compras y actualización dinámica de precios.

Funcionalidades Implementadas

  • Control de productos con fecha de caducidad
  • Alertas automáticas por vencimiento próximo
  • Registro de compras de productos
  • Reemplazo y actualización de precios desde compras
  • Edición de precios y productos (funcionalidad extendida)
  • Adaptación del sistema para farmacias
  • Configuración para tiendas mayoristas de abarrotes
  • Mejora en la impresión del ticket de venta

Cambios Visuales y Funcionales

Alertas automáticas por fecha de caducidad de productos.
Módulo de compras con actualización y reemplazo de precios.
Edición de productos y precios extendida.
Mejora en el formato del ticket de venta.

Tecnologías Utilizadas

  • PHP
  • MySQL
  • JavaScript / jQuery
  • AJAX
  • HTML5 / CSS3
